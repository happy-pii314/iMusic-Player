# 🎵 IMusic - Android Music Player

IMusic is a simple Android music player application built using **Java in Android Studio**.
The app scans the device storage and displays all MP3 songs available on the device.
Users can play music with basic player controls and a clean UI.

---

## 📱 Features

* 🎵 Play MP3 songs from device storage
* ⏯ Play / Pause music
* ⏭ Next and Previous song controls
* 🔀 Shuffle mode
* 🔁 Repeat mode
* 🎚 SeekBar with real-time song progress
* ⏱ Current time and total song duration
* 🖼 Automatic album art detection from songs
* 🎨 Custom UI with background images

---

## 🛠 Built With

* **Java**
* **Android Studio**
* **Android MediaPlayer API**
* **MediaMetadataRetriever (for album art)**
